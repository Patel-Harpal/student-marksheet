Project Title

Employee Management System

Description

This project is an Employee Management System developed using MERN Stack (MongoDB, Express, React, Node.js).
The system helps to manage employee records, salary details, and leave management efficiently.

Features

User Login and Authentication

Add New Employee

Update Employee Details

Delete Employee

Salary Management

Leave Management

Admin Dashboard

Technologies Used

MongoDB

Express.js

React.js

Node.js

Installation

Clone the repository
git clone https://github.com/your-username/project-name.git

Go to project folder
cd project-name

Install dependencies
npm install

npm start

Folder Structure

frontend – React application

backend – Node.js and Express server

database – MongoDB

Usage

After starting the server, open your browser and go to:
http://localhost:3000
